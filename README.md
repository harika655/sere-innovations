# Sere Innovations Landing Page

**Helping farmers hatch their own future.**

Sere Innovations is an agri-tech initiative dedicated to making practical technology accessible at the grassroots level. This landing page showcases our smart incubation systems designed specifically for small poultry farmers in rural and semi-urban regions.

## 🚀 Project Overview

This project was built to address the critical dependency small farmers have on commercial hatcheries. By providing an affordable, automated, and easy-to-use solution, we empower farmers to take back control of their production cycles and increase their long-term income potential.

### Key Value Propositions:
- **Affordable Pricing:** Positioned around approximately ₹10,000 to be accessible to small farm owners.
- **Ease of Use:** Designed for users without technical expertise.
- **Smart Automation:** Automated temperature, humidity, and egg-turning support to ensure high hatch rates.
- **Farmer Independence:** Reduces reliance on external hatcheries and eliminates middleman margins.

## 🎨 Design Theme: Artistic Flair

The UI follows a sophisticated "Artistic Flair" aesthetic, inspired by mid-century minimalist architecture and gallery curation:
- **Palette:** Gallery Bone (`#F2F0EB`) background, Obsidian Ink (`#121212`) text, and Ochre (`#A67C52`) accents.
- **Typography:** Elegant Serif headings (Cormorant Garamond) paired with minimalist Sans-serif body text (Inter).
- **Layout:** A clean, grid-based structure that emphasizes content hierarchy and structural autonomy.

## 🛠 Tech Stack

- **React:** For building a component-based, high-performance UI.
- **JavaScript:** Pure JS implementation for simplicity and accessibility.
- **Vite:** Modern, lightning-fast build tool and development server.
- **Tailwind CSS:** Utility-first styling for a completely bespoke design.
- **Motion (Framer Motion):** For smooth, purposeful animations and transitions.
- **Lucide-React:** For a consistent and clean iconography system.

2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Install dependencies:
   ```bash

   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
